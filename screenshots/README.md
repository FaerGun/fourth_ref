# Скриншоты интерфейса

Эта директория содержит скриншоты интерфейса проекта "Кассиопея".

## Требуемые скриншоты

1. **dashboard.png** - Главная страница Dashboard с:
   - Данными ISS (скорость, высота)
   - Интерактивной картой МКС
   - Графиками скорости и высоты
   - JWST галереей
   - Астрономическими событиями

2. **osdr.png** - Страница OSDR с:
   - Списком элементов OSDR
   - Фильтрацией и пагинацией

3. **cms.png** - Пример CMS страницы с:
   - Динамическим контентом из БД
   - CMS блоками

## Как сделать скриншоты

1. Запустите проект: `docker-compose up -d`
2. Откройте в браузере: http://localhost:8080/dashboard
3. Сделайте скриншоты страниц
4. Сохраните в эту директорию с указанными именами

## Формат

- Формат: PNG
- Разрешение: минимум 1920x1080
- Размер файла: оптимизировать для веб (можно использовать инструменты сжатия)
